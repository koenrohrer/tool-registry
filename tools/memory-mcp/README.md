# Memory MCP Server

Official MCP reference server for persistent memory via a local knowledge graph.

## What it does

Stores entities, relationships, and observations in a knowledge graph that persists across sessions. Your agent can create, query, and update structured memories.

## Use cases

- Retaining project context across long-running conversations
- Building up knowledge about a codebase over time
- Tracking decisions, preferences, and recurring patterns
- Maintaining state between separate agent sessions

## Requirements

- Node.js 18+
