# Fetch MCP Server

Official MCP reference server for retrieving and converting web content.

## What it does

Fetches web pages and converts them to clean markdown for LLM consumption. Strips navigation, ads, and boilerplate to extract the useful content.

## Use cases

- Pulling documentation pages into agent context
- Reading API references and changelogs from the web
- Summarizing articles and blog posts
- Fetching JSON from REST APIs

## Requirements

- Node.js 18+
