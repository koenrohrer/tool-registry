# Web Design Guidelines

Review UI code for compliance with Web Interface Guidelines.

## Features

- Fetches latest guidelines from Vercel web-interface-guidelines repo
- Checks accessibility, design patterns, and UX best practices
- Outputs findings in terse file:line format
- Works with any frontend file or pattern

## Usage

Ask your agent to review UI code, check accessibility, audit design, or review UX. Provide a file or glob pattern and it will fetch the latest rules and check your code against them.
