# Social Content

Comprehensive social media content creation and strategy skill.

## Features

- Platform-specific strategies for LinkedIn, Twitter/X, Instagram, TikTok, Facebook
- Content pillar framework for consistent posting
- Post templates and hook formulas for each platform
- Content repurposing system (blog to social, podcast to social)
- Content calendar planning and batching strategies
- Engagement strategy and analytics guidance

## Usage

Ask your agent to help create, schedule, or optimize social media content. It will gather context about your goals, audience, and brand voice, then produce platform-optimized content with proven engagement patterns.
