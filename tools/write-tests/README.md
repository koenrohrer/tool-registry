# Write Tests

Generates thorough unit tests for any source file. Detects your testing framework, matches existing test conventions, and covers happy paths, edge cases, and error conditions.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "Write Tests"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/write-tests [file-path]`

Example: `/write-tests src/services/auth.service.ts`
