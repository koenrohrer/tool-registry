# Scaffold Feature

Scaffolds a new feature in your project following existing conventions. Analyzes your project structure to determine naming patterns, directory organization, and testing setup, then creates source and test files that fit right in.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "Scaffold Feature"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/scaffold-feature [feature-name]`

Example: `/scaffold-feature user-authentication`
