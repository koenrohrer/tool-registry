# Explain Codebase

Generates a concise architectural overview of your project or a specific subsystem. Covers purpose, stack, directory structure, data flow, key abstractions, and entry points.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "Explain Codebase"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/explain-codebase [optional: subsystem or directory]`

Examples:
- `/explain-codebase` — overview of the entire project
- `/explain-codebase src/services` — deep dive into the services layer
