# Brave Search MCP Server

Official MCP server for privacy-first web search using the Brave Search API.

## What it does

Provides web search and local search capabilities through the Brave Search API. Returns current, high-quality results without tracking.

## Use cases

- Researching libraries, APIs, and documentation
- Finding solutions to error messages and bugs
- Looking up current events and recent releases
- Local business and location searches

## Configuration

You will be prompted for a Brave Search API key during install. Get a free key at [brave.com/search/api](https://brave.com/search/api/).

## Requirements

- Node.js 18+
- Brave Search API key (free tier available)
