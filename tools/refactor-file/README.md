# Refactor File

Analyzes a file for refactoring opportunities — long functions, duplication, deep nesting, dead code — and applies targeted improvements with your approval. Preserves public APIs and verifies tests pass.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "Refactor File"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/refactor-file [file-path]`

Example: `/refactor-file src/services/payment.service.ts`
