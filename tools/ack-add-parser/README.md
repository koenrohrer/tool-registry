# ACK Add Parser

Scaffolds a new parser for the ACK VS Code extension following the adapter pattern, error-handling conventions, and test patterns. Creates the parser file, test file, and wires it into the adapter.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "ACK Add Parser"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/ack-add-parser [parser-name]`

Example: `/ack-add-parser cursor-settings`

Designed specifically for the ACK VS Code extension codebase.
