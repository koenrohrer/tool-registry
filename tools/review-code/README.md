# Review Code

Performs a focused code review checking for bugs, security vulnerabilities, performance issues, and maintainability concerns. Reports only high-confidence findings, not style preferences.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "Review Code"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/review-code [file-or-directory]`

Example: `/review-code src/services/auth.service.ts`
