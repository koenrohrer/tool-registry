# ACK Add Service

Scaffolds a new service for the ACK VS Code extension with constructor injection, Result types, and test boilerplate. Wires it into the extension activation flow.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "ACK Add Service"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/ack-add-service [service-name]`

Example: `/ack-add-service analytics`

Designed specifically for the ACK VS Code extension codebase.
