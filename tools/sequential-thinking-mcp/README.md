# Sequential Thinking MCP Server

Official MCP reference server for structured, multi-step reasoning.

## What it does

Enables your agent to work through complex problems step by step, with the ability to revise, branch, and backtrack through thought sequences. Adds a deliberate reasoning layer before arriving at conclusions.

## Use cases

- Breaking down complex architectural decisions
- Working through multi-step debugging methodically
- Planning implementations with revision and backtracking
- Analyzing trade-offs between competing approaches

## Requirements

- Node.js 18+
