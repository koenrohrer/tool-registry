# ACK Add Tree Command

Adds a new VS Code command to the ACK sidebar tool tree. Creates the handler, registers it in `package.json` with menu contributions and `when` clauses, and adds a test.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "ACK Add Tree Command"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/ack-add-tree-command [command-name]`

Example: `/ack-add-tree-command duplicateTool`

Designed specifically for the ACK VS Code extension codebase.
