# Doc Co-Authoring

Structured workflow for co-authoring documentation with your agent.

## What it does

Guides your agent through a collaborative writing process for documentation, proposals, technical specs, and decision documents. Provides structure without being rigid -- adapts to the document type and your preferences.

## Use cases

- Writing technical specs and design documents
- Drafting project proposals and RFCs
- Creating decision documents with trade-off analysis
- Co-authoring README files and user guides
