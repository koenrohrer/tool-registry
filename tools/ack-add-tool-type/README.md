# ACK Add Tool Type

Walks through the full process of adding a new tool type to the ACK extension: enum definition, parser, writer, Zod schema, adapter routing, config service scopes, tree view group, path constants, and tests.

## Installation

Install via the [ACK extension](https://marketplace.visualstudio.com/items?itemName=koenrohrer.ack):
1. Open the ACK Marketplace panel
2. Search for "ACK Add Tool Type"
3. Click Install

## Configuration

No configuration required.

## Usage

Use with: `/ack-add-tool-type [type-name]`

Example: `/ack-add-tool-type snippet`

Designed specifically for the ACK VS Code extension codebase.
