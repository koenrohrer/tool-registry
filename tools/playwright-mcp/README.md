# Playwright MCP Server

Browser automation and testing powered by [Playwright](https://playwright.dev/), maintained by Microsoft.

## What it does

Gives your agent the ability to interact with web pages through structured accessibility snapshots -- no screenshots or vision models needed. Navigate pages, click elements, fill forms, and capture state.

## Use cases

- End-to-end testing of local web applications
- Debugging UI behavior by inspecting rendered elements
- Automating repetitive browser workflows
- Capturing screenshots for verification

## Requirements

- Node.js 18+
