# Agent Evaluation

Testing and benchmarking skill for LLM agents.

## Features

- Behavioral regression testing with statistical analysis
- Capability assessment across multiple dimensions
- Reliability metrics and production monitoring
- Adversarial testing to expose failure modes

## Usage

Ask your agent to evaluate, test, or benchmark another agent and it will apply structured evaluation frameworks with statistical rigor.
