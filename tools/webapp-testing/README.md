# Web Application Testing

Test local web applications using Playwright -- verify frontend functionality, debug UI behavior, capture screenshots, and view browser logs.

## What it does

Gives your agent a structured workflow for testing web apps: start servers, navigate pages, discover elements, execute interactions, and verify results. Includes helper scripts for managing server lifecycles.

## Use cases

- Verifying frontend functionality after code changes
- Debugging UI behavior by inspecting rendered state
- Capturing screenshots for visual verification
- Automating form submissions and user flows

## Requirements

- Python 3.10+
- Playwright (`pip install playwright && playwright install`)
