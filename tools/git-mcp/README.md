# Git MCP Server

Official MCP reference server for Git repository operations.

## What it does

Provides tools to read, search, and manipulate Git repositories. Log history, diff branches, search commits, and inspect repo state.

## Use cases

- Exploring commit history and diffs
- Searching for changes across branches
- Inspecting repository state programmatically
- Automating Git workflows through your agent

## Requirements

- Node.js 18+
- Git installed and available on PATH
