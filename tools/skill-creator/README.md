# Skill Creator

Meta-skill for creating effective Claude Code skills.

## Features

- Guided skill creation process from understanding to packaging
- Progressive disclosure design patterns
- Frontmatter and SKILL.md best practices
- Bundled resource organization (scripts, references, assets)
- Skill validation and packaging workflow

## Usage

Ask your agent to create a new skill or update an existing one. It will guide you through understanding requirements, planning resources, initializing, editing, packaging, and iterating on the skill.
