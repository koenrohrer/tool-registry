# AI Agents Architect

Skill for designing and building autonomous AI agent systems.

## Features

- Agent architecture design (ReAct, Plan-and-Execute, Tool Registry patterns)
- Tool and function calling best practices
- Agent memory systems design
- Multi-agent orchestration
- Agent evaluation and debugging guidance

## Usage

Ask your agent to help design, build, or debug an AI agent system and it will apply proven architectural patterns with guardrails for graceful degradation.
