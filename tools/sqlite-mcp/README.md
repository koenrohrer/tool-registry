# SQLite Database MCP Server

Query and manage SQLite databases through the Model Context Protocol.

## Features

- Execute SQL queries (SELECT, INSERT, UPDATE, DELETE)
- Schema inspection and table management
- Transaction support
- Read-only mode available

## Configuration

- **SQLITE_DB_PATH**: Path to your SQLite database file (default: `./data.db`)

## Requirements

- Node.js 18+ (for npx)
